RBApy is a Python package that enables creation and simulation of bacterial Resource Balance Analysis (RBA) models (https://doi.org/10.1016/j.automatica.2011.02.038), and can be interfaced to Escher maps and Proteomaps for visualisation. The inputs needed in order to create an RBA model are an annotated metabolic reconstruction in Systems Biology Markup Language (SBML) format and an Uniprot organism ID, which is used for automatic download and integration of macromolecular composition data into the model. The RBA model accurately captures the macromolecular composition of the cell, and can be refined indefinitely by adding cellular process definitions (such as translation, transcription, secretion, chaperoning etc.)
