RBApy is a Python package that enables creation and simulation of Resource Balance Analysis (RBA) models. The inputs needed in order to create an RBA model are an annotated metabolic reconstruction in SBML format and an Uniprot organism ID, which is used for automatic download and integration of macromolecular composition data into the model. The RBA model accurately captures the macromolecular composition of the cell, and can be refined indefinitely by adding cellular process definitions (such as translation, transcription, secretion, chaperoning etc.)
