Documentation

For a more detailed documentation on specific parts of RBApy, you can access the documentation available on our github pages, in the docs/ folder. There you will find more extensive documentation on:

  • Model construction (RBApy.prerba)
  • Model computation (RBApy.core)
  • Parameter estimation (RBApy.estim)
  • The XML format in which the cellular description is encoded.
